# single-bacteria-AST

This repository contains a collection of tool functions for analyzing single-cell Antibiotic Susceptibility Testing (AST) experiments.

These tools were used in the preprint:

**Phenotypic Antibiotic Susceptibility Testing at the Limit of One Bacterial Cell**\
*Irfan Ahmad, Lisa Johansson, Jimmy Larsson, Spartak Zikrin, Petter Knagge, David Fange, Johan Elf*\
bioRxiv, 2025.04.13.648565\
[DOI: 10.1101/2025.04.13.648565](https://doi.org/10.1101/2025.04.13.648565)
